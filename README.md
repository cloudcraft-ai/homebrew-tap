# CloudCraft AI — Homebrew Tap

Casks for [CloudCraft AI](https://clipbob.app) apps.

## Install

```sh
brew install --cask cloudcraft-ai/tap/clipbob
```

## Casks

| Cask | App | Description |
|---|---|---|
| `clipbob` | [ClipBob](https://clipbob.app) | Clipboard history that syncs across all your Macs |
| `rustic-rust` | [Rustic Rust](https://rustic-playground.app) | Playground for running Rust experiments, inspired by Swift Playgrounds |

Apps are signed, notarized, and auto-update via Sparkle (`auto_updates true` — Homebrew
won't fight the built-in updater; `brew upgrade` also works and keeps new installs current).
